# My Insights — Personal Blog Website

Welcome to the source code repository for **My Insights**, a personal blog website by Jivesh Bhandari. It serves as a modern thinking space at the intersection of technology, cybersecurity, engineering, and deep thinking.

🌐 **Live Demo:**: https://jivesh-bhandari-blogs.netlify.app/

---

## 🚀 Features

- **Modern Aesthetic:** A premium, minimalist dark-themed user interface featuring elegant typography (`DM Serif Display` and `DM Sans`).
- **Fully Responsive:** Fluid layouts designed with modern CSS Grid and Flexbox, optimizing the reading experience across mobile, tablet, and desktop screens.
- **Scroll Animations:** Native JavaScript `IntersectionObserver` used for lightweight, performance-friendly fade-up entry animations.
- **SEO & Social Optimization:** Pre-configured with Open Graph (OG) and Twitter Card meta tags for clean link previews when shared on social media platforms.
- **Lightweight & Fast:** Zero heavy frameworks or external dependencies—built entirely using vanilla web technologies.

---

## 🛠️ Tech Stack

- **Markup:** HTML5 (Semantic elements)
- **Styling:** CSS3 (Custom Properties/Variables, CSS Grid, Flexbox, Media Queries)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Hosting:** Netlify
